//4.what is the output?
package a;
class display 
    {
        int x;
        void show() 
        {
            if (x > 1)
                System.out.print(x + " ");
        }
    }
    class Hello 
    {
        public static void main(String args[]) 
        {
            display[] arr=new display[3];
            for(int i=0;i<3;i++)
                arr[i]=new display();
            arr[0].x = 0;      
            arr[1].x = 1;
            arr[2].x = 2;
            for (int i = 0; i < 3; ++i)
                arr[i].show();
         }
    }

a) 0
b) 1
c) 2
d) 0 1 2
Answer: c
Array creation ≠ object creation
new display[3] creates 3 references, not objects.

Every new creates one object
arr[i] = new display(); → 3 separate objects.

Instance variables belong to objects
arr[i].x = value assigns x for that specific object.

Static method can access instance data via object
main() can do obj.x — totally legal.

Apply method logic object by object
Only objects satisfying the condition produce output.

Final output comes from the condition, not the array size
